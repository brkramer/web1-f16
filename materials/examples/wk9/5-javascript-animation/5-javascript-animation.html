Javascript Animation

In addition to CSS animations, you can use Javascript to create animations on the web.

Popular options include:
  • animate with jQuery
  • Javascript animations generated from Adobe Animate CC. Formerly Adobe Flash.
  • p5js (examples).

One can also code Javascript animations directly, see Javascript Animation: Tutorial, Part 1, part 2 and part 3 by Scott Schiller.

There is also an experimental “animate” function built-in to Javascript in newer browser versions. More information at CSS Versus JavaScript Animations on Google.
